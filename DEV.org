* Basic use
** Setup
#+BEGIN_SRC sh
# may need this first:
./rebar get-deps

make
mkdir /tmp/log_air
#+END_SRC

** Running tests
: make test

** Starting the service in a shell
#+BEGIN_SRC sh
erl -pz ebin deps/*/ebin -run crdt_wal -crdt_wal actor air -crdt_wal \
    log_dir '"/tmp/log_air"' -crdt_wal data_dir '"/tmp/log_air"'
#+END_SRC
